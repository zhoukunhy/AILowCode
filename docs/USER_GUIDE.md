## 1. 系统概述

### 1.1 系统简介

AI 低代码平台是一个基于 **Monorepo 架构**的全栈低代码开发平台，集成了：

- **可视化画布编辑器**：拖拽式组件设计，所见即所得
- **AI 智能生成**：基于大语言模型的代码和组件生成
- **RAG 知识库**：检索增强生成，整合项目编码规范和历史代码
- **LangGraph 多 Agent**：多智能体协作，复杂任务分解执行

### 1.2 技术架构

| 层级 | 技术栈 | 说明 |
|------|--------|------|
| **前端** | Next.js + React + TailwindCSS | 画布编辑器 + 管理后台 |
| **后端** | NestJS + TypeORM | RESTful API 服务 |
| **数据库** | SQLite / PostgreSQL | 数据持久化 |
| **AI** | LangChain + LangGraph | AI 能力引擎 |
| **向量库** | Chroma (可选) | RAG 知识检索 |

### 1.3 系统入口

| 应用 | 地址 | 说明 |
|------|------|------|
| 画布编辑器 | http://localhost:3000 | 可视化页面设计 |
| 管理后台 | http://localhost:3001 | 系统管理 |
| API 文档 | http://localhost:3002/api/docs | Swagger 文档 |

## 2. 快速开始

### 2.1 登录系统

1. 打开浏览器访问 **http://localhost:3000**
2. 输入用户名和密码（首次使用请先注册）
3. 点击「登录」进入系统

### 2.2 默认账号

| 角色 | 用户名 | 密码 |
|------|--------|------|
| 管理员 | admin | admin123 |
| 普通用户 | user | user123 |

### 2.3 创建第一个页面

1. 进入「画布管理」页面
2. 点击「新建画布」按钮
3. 输入页面名称和描述
4. 点击「创建」进入编辑器
5. 从左侧组件面板拖拽组件到画布
6. 点击「保存」或「发布」

## 4. 使用指南

### 4.1 设计一个表单页面

**目标**：创建一个用户注册表单

**步骤**：

1. 进入「画布管理」
2. 点击「新建画布」
3. 输入名称「用户注册表单」
4. 进入编辑器

**添加组件**：

5. 从组件面板拖入「表单容器」
6. 拖入「输入框」，配置 label="用户名"
7. 拖入「输入框」，配置 label="邮箱"
8. 拖入「密码框」，配置 label="密码"
9. 拖入「按钮」，配置 text="注册"

**配置数据绑定**：

10. 选中表单容器
11. 在右侧属性面板点击「数据绑定」
12. 选择数据源和实体
13. 选择操作类型（创建）

**保存发布**：

14. 点击「保存」
15. 点击「预览」查看效果
16. 点击「发布」

### 4.2 配置数据源连接

**目标**：连接 MySQL 数据库

**步骤**：

1. 进入「数据源管理」
2. 点击「新建数据源」
3. 选择类型「MySQL」
4. 填写配置信息：

```
名称：生产数据库
主机：localhost
端口：3306
用户名：root
密码：your_password
数据库名：app_db
```

5. 点击「测试连接」
6. 提示成功后点击「保存」

### 4.3 创建知识库检索

**目标**：上传文档并用于 AI 生成

**步骤**：

1. 进入「知识库」
2. 点击「上传文档」
3. 选择文件（支持 PDF、TXT、Markdown）
4. 设置分类
5. 点击「开始向量化」
6. 等待处理完成

**使用知识库**：

7. 进入「AI 代码生成」
8. 勾选「启用 RAG」
9. 输入需求描述
10. 系统会自动检索相关知识

### 4.4 配置 Webhook 通知

**目标**：当画布发布时发送通知

**步骤**：

1. 进入「Webhook 管理」
2. 点击「新建 Webhook」
3. 填写配置：

```
名称：画布发布通知
URL：https://your-server.com/webhook
触发方式：异步
触发事件：canvas.updated
```

4. 添加签名密钥（可选）
5. 点击「创建」

**测试配置**：

6. 点击测试按钮
7. 查看日志确认通知发送成功

### 4.5 管理用户权限

**目标**：创建自定义角色并分配权限

**步骤**：

1. 进入「角色管理」
2. 点击「新建角色」
3. 输入角色名称「内容编辑」
4. 勾选权限：

```
✅ 画布管理 - 查看
✅ 画布管理 - 编辑
✅ 数据管理 - 查看
✅ 数据管理 - 新增
✅ 数据管理 - 编辑
```

5. 点击「保存」

**分配用户**：

6. 进入「用户管理」
7. 找到目标用户
8. 点击「编辑角色」
9. 选择「内容编辑」角色
10. 保存

## 5. AI 能力

### 5.1 AI 代码生成

基于自然语言描述自动生成代码。

**使用方式**：

1. 进入「AI 代码生成」页面
2. 输入需求描述，例如「创建一个用户列表页面，包含分页」
3. 选择生成类型（前端/后端/全栈）
4. 点击「生成」

**生成结果**：

- 自动生成 React 组件
- 自动生成 TypeScript 类型
- 自动生成 API 接口
- 自动生成数据库模型

### 5.2 AI 助手

智能问答助手，解答使用问题。

**使用方式**：

1. 点击右下角 AI 助手图标
2. 输入问题
3. 获取 AI 回答

### 5.3 RAG 知识检索

结合知识库的智能检索。

**配置流程**：

1. 准备知识文档（编码规范、设计模式等）
2. 进入「知识库」上传文档
3. 等待向量化处理完成

**使用场景**：

- AI 代码生成时提供上下文
- 异常诊断时检索解决方案
- 智能问答时补充知识

## 6. 常见问题

### 6.1 登录问题

**Q：忘记密码怎么办？**

A：联系系统管理员重置密码。

**Q：登录提示 CORS 错误？**

A：检查后端服务是否正常运行，确保 API 地址配置正确。

### 6.2 数据问题

**Q：保存的数据在哪里查看？**

A：进入对应模块查看，例如画布在「画布管理」查看。

**Q：如何导出数据？**

A：在数据列表页面，点击「导出」按钮，选择导出格式。

### 6.3 性能问题

**Q：画布组件太多加载慢？**

A：平台已内置虚拟化和性能优化，如仍卡顿可开启性能监控查看详情。

### 6.4 AI 功能

**Q：AI 功能无法使用？**

A：检查以下配置：
1. 系统设置中是否配置了 LLM API Key
2. 网络是否能访问 AI 服务商

**Q：RAG 检索结果不准确？**

A：可以优化知识库：
1. 上传更多相关文档
2. 调整向量模型参数
3. 使用更精确的检索策略
